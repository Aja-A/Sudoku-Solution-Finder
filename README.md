# Sudoku-Solution-Finder by Aja Agbaeze
It can find 1 of the 6,670,903,752,021,072,936,960 Sudoku solutions at random and record how long it took to find it.
You can download the .txt file and use it in your terminal.
The sudoku board is generated by filling each square indiviually and restarting if it can't fill every square correctly.
The only file is The_Sudoku_code
